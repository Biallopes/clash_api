## clash Royale 

Code to control clan to the Clash Royale game.



1. Criar ambiente virtual:
    ```
    python -m venv .venv
    cd .venv/Scripts/
    activate.bat
    ```

2. Gerar requirements
    ```
    pip freeze > requirements.txt
    ```

3. Rodar requirements local:
    ```
    pip install -r requirements.txt
    ```
